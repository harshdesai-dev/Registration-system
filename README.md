# Python Registration System 🐍

A beginner-friendly registration system built using Python basics.

## Features
- Username validation
- Password length validation
- Password must contain at least one number
- Password confirmation
- Loop until successful registration

## Concepts Used
- Variables
- if-else
- while loop
- for loop
- String handling

## Future Improvements
- Login system
- File storage
- Database integration
- GUI with Tkinter
